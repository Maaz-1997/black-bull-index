# Repository guidance

> [!IMPORTANT]
> Avoid rewriting published git history — no force-pushing, rebasing, amending, or squashing
> commits that are already pushed. Keep the working branch in a building, deployable state.

Project docs live in `docs/build-plan.md` (execution plan) and `black-bull-index.md` (product spec).
Steering rules are in `.kiro/steering/`.
